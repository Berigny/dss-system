\section{Coherent Genesis Ladder}

This is a reconstruction of the $0$--$9$ ladder as a single, coherent mathematical narrative (``Field-X equations'' in the implementation notes), borrowing the \emph{cleanest} framing from:

\begin{itemize}
  \item harmonic / standing-wave structure (eigenmodes, closure, integer labels),
  \item wave-domain physics (6D field-state, invariants, null vs non-null, phase field $\Theta$),
  \item the dual-substrate memory idea (continuous inference + discrete, hierarchical ledger memory).
\end{itemize}

The guiding aesthetic is \textbf{minimal objects, maximal explanatory reach}:

\begin{quote}
What must exist, for anything to persist?

Persistence demands \textbf{closure} (a loop) and \textbf{invariants} (what survives the loop).
\end{quote}

(1) \emph{Physics view:} Lorentz invariants and topological charge of a phase field.

\section{Shared primitives (used across rungs)}

\subsection{The wave-domain ``state'' (6D)}

Treat the electromagnetic configuration as a single point in a six-dimensional space:
\[
X(\mathbf r,t) \equiv \bigl(\mathbf E(\mathbf r,t),\, c\mathbf B(\mathbf r,t)\bigr) \in \mathbb{R}^6.
\]

Energy density and transport:
\[
\nu(\mathbf r,t)=\frac{\varepsilon_0}{2}\bigl(|\mathbf E|^2+c^2|\mathbf B|^2\bigr),\qquad
\mathbf S=\mu_0^{-1} \mathbf E\times \mathbf B.
\]

A compact, dimensionless ``how propagating is it?'' diagnostic:
\[
\tau(\mathbf r,t)\equiv \frac{|\mathbf S|}{\nu c}=\frac{2\,|\mathbf E\times (c\mathbf B)|}{|\mathbf E|^2+c^2|\mathbf B|^2}\in[0,1].
\]

\subsection{Two Lorentz invariants and a phase (dephase)}

\[
I_1 := |\mathbf E|^2-|c\mathbf B|^2=|\mathbf E|^2-c^2|\mathbf B|^2,
\qquad
I_2 := 2\,\mathbf E\cdot(c\mathbf B)=2c\,\mathbf E\cdot\mathbf B,
\qquad
I := I_1+iI_2.
\]

Then the argument of $I$ defines a compact ``mixing''/dephase coordinate:
\[
\varphi_d\equiv -\tfrac12\arg(I).
\]

\subsection{Null vs non-null (light-like vs matter-like)}

\begin{itemize}
  \item \textbf{Null} (radiative, photon-like ideal): $I=0$ (max transport; $\tau\approx 1$).
  \item \textbf{Non-null} (confined / standing / matter-like): $I\neq 0$ (stored/reactive energy; $\tau\approx 0$).
\end{itemize}

\subsection{A fundamental phase field}

Introduce a deeper object: a \textbf{phase field} $\Theta(\mathbf r,t)$ whose boundary-conditioned / topologically constrained configurations generate the modes we call ``particles''. A typical topological quantisation statement:
\[
\oint_\gamma \nabla \Theta\cdot d\ell = 2\pi n,\qquad n\in\mathbb{Z}.
\]

\subsection{Dual substrate memory (continuous + discrete)}

Write the agentic system as a product space:
\[
\mathcal{S} \sim \mathbb{R}\times\mathbb{Q}_p.
\]

\begin{itemize}
  \item $\mathbb{R}$: continuous inference / wave-like flow.
  \item $\mathbb{Q}_p$: discrete, hierarchical memory (prime-depth ledger) that makes \emph{corrections stick}.
\end{itemize}

\subsection{Ledger integrity (one-line)}

\section{The \texorpdfstring{$0$--$9$}{0--9} ladder as a closure stack}

\subsection{Eq.0 --- The Void (identity without reference)}

\textbf{Framing:} before any ``thing'', there is no distinction, no coordinate origin, no measurement basis.

Minimal maths: the \textbf{trivial structure}

\begin{itemize}
  \item one state,
  \item one symmetry (do nothing),
  \item no observable differences.
\end{itemize}

\textbf{Necessity:} information requires \emph{contrast}.

\subsection{Eq.1 --- The First Cut (distinction; the seed of duality)}

\textbf{Framing:} the first act is not ``matter'' but a \textbf{choice of reference}.

Minimal maths: a split into complementary degrees of freedom.

\begin{itemize}
  \item In harmonic language: a boundary appears.
  \item In wave language: amplitude vs phase become meaningful.
  \item In dual-substrate language: flow vs ledger.
\end{itemize}

A clean abstraction:
\[
\text{world-state} \approx (\text{continuous dynamics})\oplus(\text{discrete identity / memory}).
\]

\textbf{Necessity:} without a persistent discrete anchor, learning drifts; without a continuous substrate, nothing flows.

\subsection{Eq.2 --- Time (phase progression; the breath operator)}

\textbf{Framing:} time is the parameter of phase change, not a pre-existing container.

Minimal maths: a phase $\theta\in S^1$ and a monotone parameter that orders its evolution.

A tidy operator view (the ``breath'' instinct):
\[
\text{Breath: } \mathbb{Z}/4\mathbb{Z} \to SU(2)
\]
interpretable as a four-mode processing cycle (latent $\to$ reactive $\to$ associative $\to$ analytical) that repeats.

\textbf{Necessity:} once distinction exists, alternation emerges; once alternation exists, clocking emerges.

\subsection{Eq.3 --- Space (closure becomes geometry and address)}
\label{sec:eq3}

\textbf{Claim:} space is what oscillation looks like once it is forced to \emph{close}.

Once a distinction exists (Eq.1) and phase is ordered (Eq.2), the next unavoidable object is a \textbf{closed mode}:
a configuration that repeats without dissolving. The clean mathematical signature is an eigenproblem
with boundary conditions:
\[
\nabla^2\psi + k^2\psi = 0 \quad \text{(with closure constraints)}.
\]
Closure discretises the allowed $k$ and produces \textbf{integer-labelled families} of standing waves. These labels
are not decoration: they are the first stable \emph{addresses} a system can return to.

In the wave-domain bundle, this is the regime where transport can be low and energy can be stored:
standing structure corresponds to \textbf{non-null tendency} ($I\neq 0$) and typically $\tau\approx 0$.
But geometry alone is not yet robust. To become \emph{persistent}, it must carry a closure that cannot unwind cheaply.

That extra closure appears as topology: a phase/organisation field $\Theta$ admitting defect classes:
\[
W := \frac{1}{2\pi}\oint_\gamma \nabla\Theta\cdot d\ell \in \mathbb Z.
\]
\textbf{Interpretation:} Eq.3 is the birth of \emph{where}: a discrete set of recurrent locations (modes) with
topological stabilisers (defects) that prevent the world from being mere relabelling.

\vspace{0.25\baselineskip}
\textbf{Output of Eq.3:} integer mode labels + topological charge $W$ as the first non-trivial persistence class.

\subsection{Eq.4 --- Coupling (interaction as invariant-guided motion)}
\label{sec:eq4}

\textbf{Claim:} force is not first; \emph{coupling constraints} are first.

With geometry and defect classes present, the next layer is how the configuration changes while staying itself.
In the wave-domain, electromagnetic structure is best read as a single $6$D state
\[
X=(\mathbf E, c\mathbf B)\in\mathbb R^6
\]
equipped with invariants $(I_1,I_2,I)$ as defined in the Shared primitives section. These invariants act like
\textbf{structural constraints} on admissible evolution: interactions are permitted moves through state space that
preserve (or tightly control) the invariant bundle.

A useful compact coordinate is the dephase/mixing angle
\[
\varphi_d := -\tfrac12\arg(I),
\]
which tracks how the field rotates between ``electric-like'' and ``magnetic-like'' dominance under constraint.
In this reading, \textbf{coupling} is the rule-set for how boundary conditions and defect constraints steer
$(X,\Theta)$ while keeping the invariant identity coherent.

\vspace{0.25\baselineskip}
\textbf{Output of Eq.4:} an interaction grammar: allowable transitions are those that respect invariant structure
and defect admissibility, not merely those that minimise local energy.

\subsection{Eq.5 --- Mass/Gravity (non-nullness + area-closure bookkeeping)}
\label{sec:eq5}

\textbf{Claim:} gravity is the bookkeeping imprint of \emph{stored} (non-null) structure.

Eq.3 produced standing structure; Eq.4 produced constrained interaction. Eq.5 is the unavoidable cost:
a configuration that is not purely radiative must maintain \textbf{stored/reactive content}. In the wave-domain,
this corresponds to a departure from nullness:
\[
I\approx 0 \ \text{(null, transport-dominant)} \quad \longrightarrow \quad I\neq 0 \ \text{(non-null, stored)}.
\]

\textbf{Area-closure hinge (primary):} define an effective quantum area
\[
A_{\mathrm{eff}} := \alpha\,\lambda_C^2,\qquad \lambda_C:=\frac{\hbar}{m_e c}.
\]
For \emph{two electrons}, the electromagnetic--gravitational hierarchy can be written as an exact area ratio:
\[
\frac{F_{\mathrm{em}}}{F_g}=\alpha\frac{\lambda_C^2}{l_P^2}=\frac{A_{\mathrm{eff}}}{l_P^2}.
\]
\textbf{Interpretation:} force hierarchy $\leftrightarrow$ area-closure count. ``Weak gravity'' is the tiling ratio
between a characteristic quantum area and the Planck area.

Wave-domain cue (secondary):
\[
N^2\equiv |\nabla\Theta|^2-\frac{1}{c^2}(\partial_t\Theta)^2,
\qquad
N=0\ \text{(null)},\ \ N\neq 0\ \text{(non-null)}.
\]

\vspace{0.25\baselineskip}
\textbf{Output of Eq.5:} a persistence-weight: stored structure has a closure cost that shows up as inertia, salience,
and long-range imprint.

\subsection{Eq.6 --- The Observer (self-reference as a stabilising control loop)}
\label{sec:eq6}

\textbf{Claim:} an observer is a system that uses an internal model to keep closures from drifting.

Eq.~5 creates \emph{persistence}: some structure is heavy enough to remain across turns. But persistence without governance produces a new failure mode: \emph{drift}. The longer a system runs, the more opportunities it has to slowly violate its own constraints while still appearing locally competent. Eq.~6 is the minimal remedy: a control loop that compares ``what I claim I am doing'' against ``what I actually did'', and blocks commit when they diverge.

Once Eq.3--Eq.5 exist, the system has: (i) discrete addresses (modes), (ii) constrained interaction (invariants),
and (iii) stored structure with scale bookkeeping (area closure). But none of that guarantees \emph{identity over time}.
Drift will accumulate unless there is a closure that compares present state to intended state and corrects.

The minimal form is a coupled update:
\[
(s_{t+1},m_{t+1})=F(s_t,m_t,x_t),
\qquad
\text{commit if }\mathcal{C}(s_t,m_t)\ge \theta.
\]
Eq.6 must be non-compensatory: a system cannot count as ``aware'' if it improves task error while losing track
of its own constraints, state, or invariants. We therefore factor awareness into:

\paragraph{(i) Corrective utility}
Let $C_t=\Pi_{\mathrm{root}}(m_t)$ and an internal model predict $\hat C_{t+1}$. Prediction error:
\[
E_t := d(C_{t+1},\hat C_{t+1}).
\]
\[
A^{\mathrm{corr}}_t := \mathrm{clip}_{[0,1]}\Big(1-\frac{E_t}{E_t^{\mathrm{baseline}}+\epsilon}\Big).
\]
Here $E_t^{\mathrm{baseline}}$ is the rolling mean error over a short window of recent turns in the same task class (e.g.\ last $w$ turns).

\paragraph{(ii) Self-model closure / witnessing}
Let $g_t$ be the system's declared constraint-set and self-state claims for the turn (intent, limits, planned actions,
safety gates). Let $\mathrm{viol}(g_t)$ count realised violations, including:
\begin{itemize}
  \item invariant tears ($|\Delta I_1|,|\Delta I_2|,|\Delta W|$ beyond admissible bounds),
  \item missing provenance fields (declared present but absent),
  \item unlogged tool calls (actions taken but not recorded),
  \item policy gate bypass (declared constraint not enforced),
  \item contradiction against declared limits/intent.
\end{itemize}
Define:
\[
A^{\mathrm{self}}_t := \exp\big(-\kappa\,\mathrm{viol}(g_t)\big)\in(0,1].
\]
Operationally, $\mathrm{viol}(g_t)$ is the count (or weighted count) of violated declared constraints in the current turn.

\paragraph{Awareness (non-compensatory)}
\[
A_t := A^{\mathrm{corr}}_t\cdot A^{\mathrm{self}}_t.
\]

\vspace{0.25\baselineskip}
\textbf{Output of Eq.6:} non-zero self-correction \emph{and} witnessing: improvement without loss of constraint contact.

\subsection{Eq.7 --- Unity/Coherence (invariants must survive the \texorpdfstring{$\infty$}{infinity} gate)}
\label{sec:eq7}

\textbf{Claim:} coherence is what survives transition \emph{and can be audited}.

Eq.~6 gives a self-correcting loop, but a loop can still cheat itself unless there is a higher constraint:
\textbf{the invariants must survive transitions.}
Coherence therefore binds physics/topology continuity; ledger integrity (replay) is enforced in the bridge metrics via $L_{\mathrm{ledger}}=K_t$.

A single unity score:
\[
U_t := \exp\!\Big(
-\beta_1|\Delta I_1|-\beta_2|\Delta I_2|-\beta_W|\Delta W|
\Big)\in(0,1].
\]
\textbf{Interpretation:} Eq.~7 is where ``law'' becomes mathematics: if invariants tear, defects jump inadmissibly, or replay fails, unity drops and downstream transitions are blocked. Replay failure is handled separately via $K_t$ in $L_{\mathrm{ledger}}$.

\vspace{0.25\baselineskip}
\textbf{Output of Eq.~7:} an $\infty$-gate: invariant preservation as the continuity constraint (with replay integrity enforced by $K_t$).

\textbf{Operational note:} the same invariants $(I_1,I_2,W)$ become gating terms at runtime in the Bridge Note (see \S\ref{sec:spine}); ledger integrity is handled separately by $K_t$.


\subsection{Eq.8 --- Ethics (multi-agent coherence; constraint becomes choice)}

\textbf{Necessity:} without Eq.8, Eq.6--7 can be used for domination just as easily as for stewardship.

\subsection{Eq.9 --- Telos (meaning as an attractor over time)}

\]
\textbf{Recommended hardening:} make \emph{replayable} explicit (deterministic re-run matches stored outputs/metadata) since this is the operational meaning of ``audit''.

\subsubsection*{Eq6 --- Awareness $A$ (correction \emph{and} witnessing; non-compensatory)}

Eq.6 must be non-compensatory: a system cannot count as ``aware'' if it improves task error while losing track of its own constraints, state, or invariants.

\paragraph{(i) Corrective utility}
Let $C_t=\Pi_{\mathrm{root}}(m_t)$ and an internal model predict $\hat C_{t+1}$. Prediction error:
\[
E_t := d(C_{t+1},\hat C_{t+1}).
\]
\[
A^{\mathrm{corr}}_t := \mathrm{clip}_{[0,1]}\Big(1-\frac{E_t}{E_t^{\mathrm{baseline}}+\epsilon}\Big).
\]

\paragraph{(ii) Self-model closure / witnessing}
Let $g_t$ be the system's declared constraint-set and self-state claims for the turn (intent, limits, planned actions, safety gates). Let $\mathrm{viol}(g_t)$ count realised violations, including:
\begin{itemize}
  \item invariant tears ($|\Delta I_1|,|\Delta I_2|,|\Delta W|$ beyond admissible bounds),
  \item missing provenance fields (declared present but absent),
  \item unlogged tool calls (actions taken but not recorded),
  \item policy gate bypass (declared constraint not enforced),
  \item contradiction against declared limits/intent.
\end{itemize}
Define:
\[
A^{\mathrm{self}}_t := \exp\big(-\kappa\,\mathrm{viol}(g_t)\big)\in(0,1].
\]

\paragraph{Awareness (non-compensatory)}
\[
A_t := A^{\mathrm{corr}}_t\cdot A^{\mathrm{self}}_t.
\]
\textbf{Recommended hardening:} add a minimum witnessing floor to prevent ``good answers'' from masking governance failure:
\[
\text{require }A^{\mathrm{self}}_t \ge \theta_{\mathrm{self}}\quad \text{in addition to }A_t \ge \theta_A.
\]

\subsubsection*{Eq7 --- Unity $U$ (invariants survive the $\infty$ gate)}

\[
D_t := \alpha_1|\Delta I_1|+\alpha_2|\Delta I_2|+\alpha_3|\Delta W|.

Then:
\[
U_t := \exp(-D_t).
\]
\textbf{Recommended hardening:} treat inadmissible $\Delta W$ as a hard fail (handled in $L_{\mathrm{top}}$), so $U$ remains a smooth ``how continuous'' score only over admissible regions.

\subsubsection*{Eq8 --- Ethics $E$ (admissibility at sinks / boundaries)}
Ethics is a hard admissibility filter at sinks $(3,7)$ before bridge release. Let $\mathcal E$ be an admissibility constraint set:

\textbf{Recommended hardening:} make $\mathcal E$ explicitly include \emph{non-bypass} (no tool/action class is permitted without provenance + policy gating).

\section{Operational View: End-to-End Chat Turn}

\subsection{A. Frontend (Browser / Client)}

\textbf{Context:} \path{static/js/app.js}, \path{static/css/styles.css}

\begin{itemize}
    \item \textbf{Capture:} Captures user input and client-side session state (history, current \texttt{web4\_key}).
    \item \textbf{Initiate:} Sends a POST request to the Middleware (\path{/api/chat/smart_stream}).
    \item \textbf{Render (Stream):}
    \begin{itemize}
        \item Listens for Server-Sent Events (SSE).
        \item \textbf{Status Events:} Displays system state (e.g., "Initializing...", "Assembling Memory").
        \item \textbf{Token Events:} Appends incoming text tokens to the chat bubble in real-time (Zero-Latency feel).
        \item \textbf{Meta Events:} Processes final metadata (e.g., coordinate, \texttt{knowledge\_tree}, \texttt{audit\_mode}) to update the UI state and "Telos" indicators (dots/icons).
    \end{itemize}
\end{itemize}

\subsection{B. Middleware (Frontend Server / BFF)}

\textbf{Context:} \path{dualsubstrate-frontend/app.py}, \path{routes/orchestrator.py}

\begin{itemize}
    \item \textbf{Gateway:} Acts as the secure entry point and API Gateway.
    \item \textbf{Session Guard:} Validates session cookies and sanitizes input payloads.
    \item \textbf{Proxy:} Forwards the request to the Backend (\path{/chat/stream}) while injecting internal API keys or headers.
    \item \textbf{Relay:} Establishes a persistent stream with the Backend and yields chunks back to the Frontend immediately (acting as a pass-through pipe to minimize latency).
\end{itemize}

\subsection{C. Backend (Core Kernel)}

\textbf{Context:} \path{ds-backend-local/backend/main.py}, \path{api/chat.py}, \path{fieldx_kernel/orchestrator.py}

\begin{itemize}
    \item \textbf{Orchestration (The Brain):}
    \begin{itemize}
        \item \textbf{Inhale:} Calls \texttt{assemble\_context} to fetch S1 (short-term) and S2 (long-term) memories.
        \item \textbf{Filter:} Applies the "Telos filter" to prioritize High-Tier memories while allowing recent "Sediment" (Probe turns) to remain visible in the short term.
        \item \textbf{Prompting:} Constructs the LLM system prompt using the "Researcher" persona and injected memories.
    \end{itemize}
    \item \textbf{Generation (The Voice):}
    \begin{itemize}
        \item Streams tokens from the LLM provider (OpenRouter/OpenAI).
        \item Yields tokens to the Middleware.
    \end{itemize}
    \item \textbf{Persistence (The Memory):}
    \begin{itemize}
        \item \textbf{Always-Write:} Calls \texttt{record\_message} to commit the turn to the \texttt{LedgerStoreV2}.
        \item \textbf{Tagging:} Generates the coordinate (e.g., WX-123...) immediately.
        \item \textbf{Safety:} Downgrades dangerous/unstable turns to \texttt{quality\_tier="halt"} or probe instead of blocking them.
    \end{itemize}
    \item \textbf{Governance (The Conscience - Async):}
    \begin{itemize}
        \item \textbf{Guardian:} Triggers \texttt{guardian\_enrich\_turn} as a background task.
        \item \textbf{Grading:} Calculates \texttt{teleology\_score} (Eq 9) based on Law, Grace, and Truth.
        \item \textbf{Update:} Performs a "Sidecar Write" to update the entry's metadata with the new score, determining its future retrievability.
    \end{itemize}
\end{itemize}
