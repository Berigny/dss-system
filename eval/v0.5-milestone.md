# DSS-EVAL v0.5 Milestone

**Milestone:** benchmark-validity v0.5  
**Pinned at:** 2026-07-20  
**Repo:** `Berigny/dss-system`  
**Tracking issue:** [dss-system/issues/1](https://github.com/Berigny/dss-system/issues/1)

## Purpose

This milestone closes the public validity critique raised in issue #1: that the
original DSS benchmarks encoded relevance in router-visible coordinates.  The
v0.5 suite replaces or augments those benchmarks with label-blind ingestion,
adversarial-integrity tests, deterministic citation checks, and a real-data
track, all reproducible from a single container command.
